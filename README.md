# PTA pintia 学习助手

> Pintia (PTA) 多功能学习助手 — AI 自动解题 + 答案检查，支持 DeepSeek / MiMo / GPT 多平台

## ✨ 功能特性

- 🤖 **AI 自动解题** — 自动提取题目、调用 AI 生成答案、填写并提交
- ✅ **答案检查模式** — 检查已作答题目，只显示错题及正确答案与解析
- 📝 **多题型支持** — 判断题、单选题、多选题、填空题、函数题、编程题
- 🚫 **突破粘贴限制** — 多策略代码填入，绕过 PTA 复制粘贴屏蔽机制
- 🖱️ **浮动面板** — 可最小化为小图标，不干扰做题
- 🌐 **多平台切换** — DeepSeek / MiMo / GPT / 自定义，一键切换
- 📦 **单题模式** — 可选择只解答或检查某一道题
- ⏭️ **自动跳转** — 答完当前题型后自动切换到下一题型
- 📋 **错误总结** — 检查完成后汇总所有错题，支持点击跳转到对应题目
- 🔍 **答案解析** — 错题自动展示 AI 解析，可折叠展开

## 📦 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 Tampermonkey 图标 → **创建新脚本**
3. 将 [`pintia-helper.js`](./pintia-helper.js) 的内容粘贴进去，保存
4. 访问 [pintia.cn](https://pintia.cn) 的考试页面，右下角出现小图标即安装成功

## 🚀 使用方法

### 基本配置

1. 打开 PTA 考试页面，点击右下角小图标展开面板
2. 切换到 **「API设置」** 标签
3. 选择平台并填写配置：

| 平台 | API URL | 默认模型 |
|------|---------|----------|
| DeepSeek | `api.deepseek.com/v1/chat/completions` | `deepseek-chat` |
| MiMo (小米) | `api.xiaomimimo.com/v1/chat/completions` | `mimo-v2.5-pro` |
| GPT | `api.openai.com/v1/chat/completions` | `gpt-4o` |
| 自定义 | 手动填写 | 手动填写 |

4. 填好 **API Key**，即可使用

### 自动解题

1. 确保当前为「自动答题」模式（顶部模式切换按钮）
2. 在 PTA 考试页面切换到要解答的题型标签
3. （可选）在题号输入框输入数字，只解答指定题目
4. 点击 **「开始答题」**
5. 等待 AI 分析题目并自动填写答案

### 检查答案

1. 切换到「检查答案」模式
2. （可选）输入题号，只检查指定题目
3. 点击 **「检查答案」**
4. 日志中只显示答错的题目，附带正确答案与解析
5. 检查完成后底部显示错误题目总结，可点击跳转

### 停止解题

- 点击 **「停止」** 按钮即可中断当前流程

## 📋 支持的题型

| 题型 | 说明 | 自动答题 | 检查答案 |
|------|------|:--------:|:--------:|
| 判断题 | TRUE_OR_FALSE | ✅ | ✅ |
| 单选题 | MULTIPLE_CHOICE | ✅ | ✅ |
| 多选题 | MULTIPLE_CHOICE_MORE_THAN_ONE_ANSWER | ✅ | ✅ |
| 填空题 | FILL_IN_THE_BLANK | ✅ | ✅ |
| 函数题 | CODE_COMPLETION | ✅ | ⚠️ 手动验证 |
| 编程题 | PROGRAMMING | ✅ | ⚠️ 手动验证 |

## ⚙️ 配置说明

### API 配置

| 配置项 | 说明 |
|--------|------|
| 平台选择 | DeepSeek / MiMo / GPT / 自定义 |
| API Key | API 密钥，点击平台后显示对应申请链接 |
| API URL | 自定义 API 地址（仅自定义模式需填写） |
| 模型名称 | 自定义模型名称（仅自定义模式需填写） |

### PTA 专属配置

| 配置项 | 说明 |
|--------|------|
| 函数题语言 | C / C++ / Java / Python |
| 编程题语言 | C / C++ / Java / Python |
| 去除代码注释 | 自动移除 AI 生成代码中的注释（适配提交要求） |
| 答完自动切换下一题型 | 完成当前题型后自动跳转到下一题型 |
| 显示答案解析 | 错题是否展示 AI 解析内容 |

### 单题模式

在底部题号输入框输入数字后点击按钮：
- **留空**：解答/检查当前题型下的所有题目
- **输入数字**：只解答/检查指定的那一道题

### 检查模式特性

- 自动跳过未作答的题目，不显示
- 正确题目仅显示 ✓，不展开解析
- 错误题目显示 ✗ + 你的答案 + 正确答案 + 解析
- 检查完成后在底部汇总所有错题（题号 + 正确答案）
- 点击错题可跳转到页面对应位置

## 🔧 API 兼容性

支持任何兼容 OpenAI Chat Completions 格式的 API：

```
POST /v1/chat/completions
{
  "model": "模型名称",
  "messages": [{"role": "user", "content": "题目内容"}],
  "temperature": 0.7,
  "max_tokens": 4096
}
```

推荐的 API 服务：
- [DeepSeek](https://platform.deepseek.com/) — 国内稳定，性价比高
- [小米 MiMo](https://platform.xiaomimimo.com/) — 国产大模型
- [OpenAI](https://platform.openai.com/) — GPT 系列

## ❓ 常见问题

**Q: 右下角小图标没有出现？**
A: 确认你在 `pintia.cn` 域名下的考试页面，Tampermonkey 已启用，尝试刷新页面。

**Q: 提示"未找到代码编辑器"？**
A: 页面未完全加载，等待加载完成后刷新重试。

**Q: AI 返回结果异常？**
A: 检查 API Key 是否正确、额度是否充足。打开 F12 控制台查看是否有报错信息。

**Q: 如何停止自动解题？**
A: 点击面板上的「停止」按钮。

**Q: 编程题代码填入不完整？**
A: 脚本使用多策略填入（CodeMirror API → execCommand → 逐行插入）。如仍有问题，可手动 Ctrl+V 粘贴（脚本会自动接管粘贴流程）。

**Q: 支持哪些编程语言？**
A: C、C++、Java、Python，可在设置中选择。
